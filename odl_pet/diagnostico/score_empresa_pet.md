# Score da Empresa — ODL Pet

## Função deste documento

Registro individual do Score ODL Pet para um cliente específico. Preenchido após a reunião de diagnóstico, com base nas evidências coletadas na preparação e nas respostas da reunião.

---

## Como preencher

1. Ler [score_odl_pet.md](../metodo/score_odl_pet.md) para os critérios detalhados de cada pilar
2. Para cada pilar, atribuir nota baseada em evidências documentadas
3. Registrar qual evidência justifica cada nota — nunca pontuação intuitiva
4. Calcular total e interpretar

## Como usar o score na apresentação ao cliente

O score não é uma nota de desempenho. É um mapa de onde o negócio está e onde pode chegar.

**Como apresentar:**
> "O score de vocês é [X]/100. Isso coloca o [nome] no estágio de [classificação]. Não é uma nota — é uma fotografia do que está estruturado hoje. O que vou propor vai mover isso de [X] para [Y] nos pilares que têm maior impacto no crescimento."

**O que NÃO fazer:**
- Usar o score para diminuir o trabalho do empresário
- Apresentar como algo negativo — é uma oportunidade, não uma crítica
- Mentir a nota para cima pra não constranger — a honestidade é o que diferencia
